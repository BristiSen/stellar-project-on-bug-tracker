# 🐞 Bug Tracker Smart Contract (Soroban)

## 📌 Project Description
This project is a decentralized Bug Tracker built using Soroban smart contracts on the Stellar blockchain. It allows users to report, manage, and track bugs in a transparent and tamper-proof manner.

## ⚙️ What it does
The smart contract enables users to:
- Create bug reports with a unique ID
- Store bug details on-chain
- Update bug status (open/closed)
- Retrieve individual or all bug reports

All data is securely stored on the blockchain, ensuring transparency and immutability.

## ✨ Features
- 🐞 Create bug reports with description
- 🔐 Secure on-chain storage
- 📊 Track bug status (open/closed)
- 📋 List all bugs
- 👤 Reporter authentication for bug creation
